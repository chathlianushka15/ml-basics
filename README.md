## What I Built

A complete beginner ML pipeline that classifies iris flowers into 3 species using just their petal and sepal measurements - trained, tested, and visualized end to end.

---

## Concepts I Actually Learned

| Concept | What It Means                                    |
|---------|--------------------------------------------------|
| Features | The inputs - measurements the model uses to decide |
| Labels | The answer  species the model tries to predict   |
| Train/Test Split | 80% to learn from, 20% hidden to honestly test on |
| `.fit()` | The actual training step  model studies patterns |
| `.predict()` | Model guesses on data it's never seen            |
| Accuracy | % of test guesses that were correct              |
| Decision Tree | A flowchart of yes/no questions the model learned |

---

## Results

Training flowers : 120

Testing flowers  : 30

Accuracy         : 100%

The model correctly identified all 30 test flowers it had never seen during training.

---

## The Decision Tree

The model learned that a single question - **"Is petal length ≤ 2.45 cm?"** - perfectly separates Setosa from the other two species. Everything else follows from there.
